## 2. Is semantics *only* the hypergraph?

**No — dual (actually triple) representation.**

| Axis | Carrier | Role |
|---|---|---|
| Semantic similarity / seed | Embedding | "Is this near that?" — **SEED** (ML17) |
| Semantic binding / logic | Hypergraph n-ary edges | "What is jointly asserted?" — **EDGE** structure |
| Semantic strength / currency | \(W(e\mid c)\) | "How strong / live / superseded?" — **Weight** |

```
semantic ≠ hypergraph alone
semantic ≠ embedding alone
Semantic Weight Map =
  embedding (coords)
  + hypergraph (n-ary circuit)
  + W (semantic strength)
  + a (activation)
```

- Hypergraph = **logical/combinatorial layer** of meaning, not the entire meaning.
- Dropping embeddings loses the empirically dominant lift body (~aligned coordinates).
- Dropping \(W\) loses HSWM identity (becomes unweighted structure).
